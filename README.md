## Bluey Memory Game
An interactive web-based game designed to enhance memory and cognitive skills.

## Technologies

- **Ruby on Rails**: Framework for building the web application.
- **SCSS**: For Design
- **Cloudinary**: For managing image uploads and storage.

## Versions Utilized

- **Ruby**: 3.x
- **Rails**: 7.x
- **Node.js**: 18.x
- **npm**: 10.x
